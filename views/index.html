Timestamp Microservice

This is a microservice API that converts a date string to a unix timestamp and vice versa, made as my submission for the Timestamp Microservice project in freeCodeCamp's Back End Development and APIs Course.

Example Usage:

/api/2015-12-25 /api/Fri, 25 Dec 2015 /api/1451001600000

Example Output:

{"unix":1451001600000, "utc":"Fri, 25 Dec 2015 00:00:00 GMT"}
